# Fonti.dev

Fonti.dev is my personal portfolio and blog where I share my work, thoughts, and experiences in web development and beyond. The site is designed to be a comprehensive showcase of my skills and projects, as well as a platform for sharing insights and stories.

## Key Features
- Personal blog with articles on web development and personal experiences
- Portfolio showcasing selected projects
- Responsive design for optimal viewing on all devices

## Tech Stack
- **Framework**: Next.js 15 (React)
- **Language**: TypeScript
- **Styling**: TailwindCSS
- **Deployment**: Vercel
- **Content Management**: Content Collection
