# CCTV Kota Malang

A modern web application that revolutionizes CCTV monitoring in Malang city by offering multi-camera viewing capabilities with an intuitive user interface. Unlike the official system, CCTV Kota Malang enables users to simultaneously view multiple CCTV feeds with customizable layouts and persistent settings.

## Key Features
- Multi-camera simultaneous viewing
- Drag-and-drop camera arrangement
- Customizable grid layout system
- Interactive map and street-based camera selection
- Local storage for user preferences
- Responsive design for various devices

## Tech Stack
- **Framework**: Next.js 14 (React)
- **Language**: TypeScript
- **Styling**: TailwindCSS
- **UI Components**: Shadcn UI (Radix UI)
- **State Management**: Zustand
- **Runtime**: Bun
- **Deployment**: Vercel
